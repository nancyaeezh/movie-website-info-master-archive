# movie-website-info-master-archive
本仓库用于集中整理和维护互联网上已公开可访问的影视类网站基础信息，重点对电影与电视剧相关站点进行系统化归档。记录内容包括站点名称、访问地址、站点定位说明及基础分类信息，便于集中管理与长期查询。本项目不涉及任何影视资源的存储、播放、下载或解析，仅对站点公开信息进行整理与留存。所有资料均来源于公开网络环境，整理过程保持中立与客观，不对第三方站点的内容质量、合法性或服务稳定性作出任何评价，也不构成任何形式的推荐或引导。
